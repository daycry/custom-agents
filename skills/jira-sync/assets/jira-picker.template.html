1 · Proyecto o issue

Si pegas una clave/URL de issue, lo uso directamente como padre. Si escribes texto, busco proyectos.

2 · ¿Cómo agrupar las tareas en Jira?

Las tareas del plan pueden agruparse bajo una épica, colgar de un issue existente, o quedar sueltas. Para una iniciativa lo normal es agruparlas en una épica.
➕ Crear una épica nueva y agrupar en ella las tareas  recomendado
○ Dejarlas sueltas en el proyecto, sin agrupar bajo ninguna épica ni tarea
Cada tarea será un issue independiente en el backlog. Útil solo si no necesitan agruparse.
…o colgar de un issue existente: si es una épica → se crean Tareas; si es una tarea/historia → se crean Subtareas. Búscalo por clave (DM5985-59), texto o JQL (issuetype = Epic).
Vacío = issues recientes del proyecto.
